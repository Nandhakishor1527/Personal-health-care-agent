Project Overview
The Personal Health Care Agent project aims to create a simple and user-friendly system that allows users to track their health and wellness. The system will have the following features:

User registration and login
User profile management
Health data tracking (weight, height, blood pressure, etc.)
Health recommendations based on user data
Project Structure
The project will be structured into the following modules:

user.py: User registration, login, and profile management
health.py: Health data tracking and recommendations
agent.py: Main application logic
